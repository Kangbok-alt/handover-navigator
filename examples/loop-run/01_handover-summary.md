## 1페이지 핵심 요약

박지훈 BM이 인수인계 없이 떠나 노바린(한빛제약 고지혈증 치료제, 가상) 브랜드 업무를 재구성했다 (출처: 01_브랜드플랜, 03_KOL). **가장 급한 것은 마감이 이미 지난 두 건의 규제/예산 건이다.** ① 약가 인하 대응자료의 심평원 제출 마감(5/29)과 약가팀 전달 마감(5/22)이 이미 경과했으며 전달 완료 여부가 확인되지 않는다 — 놓쳤다면 약가 직접 타격 (출처: 06_규제, 05_이메일, 04_업무메모). ② 4분기 예산 재배분 건은 본부장이 보완 요청 후 회신이 없는 채로 멈춰 있다 (출처: 05_이메일, 04_업무메모). 그 외 메디컴(광고대행사) 대금 정산 (출처: 03_KOL, 04_업무메모, 05_이메일), 디테일 자료 v3 광고심의 (출처: 06_규제), 하반기 심포지엄 대체 연자 확보 (출처: 03_KOL)가 미결로 남아 있다. 공유드라이브 권한 이관도 신청되지 않아 후임자 접근 자체가 막혀 있을 수 있다 (출처: 04_업무메모, 05_이메일).

**첫 주 최우선 3가지**
1. 약가 대응자료가 심평원에 제출됐는지 즉시 확인 (→ 약가팀 최약가 차장) (출처: 06_규제, 05_이메일, 03_KOL / 미결 #1)
2. 공유드라이브 등 계정 권한 신청 (→ IT 헬프데스크 양식) (출처: 04_업무메모, 05_이메일 / 미결 #6)
3. 메디컴 대금 정산 처리 여부 확인 (→ 경리/정대행 부장) (출처: 03_KOL, 04_업무메모, 05_이메일 / 미결 #3)

## 블록 5 — 맥락(Why) / 배경

- **약가 방어가 브랜드 전략의 축**: 노바린이 약가 재평가 대상에 포함돼, 약가 인하 방어가 2026 전략 3대 축 중 하나 (출처: 01_브랜드플랜 전략축 3, 06_규제). 약가 변수가 연 타겟 달성의 최대 변수로 인식됨 (출처: 02_월간실적 누적메모).
- **가정의학과 확대가 성장 동력**: 순환기는 안정적이나 가정의학이 약해, Q2 가정의학 드라이브 성패가 연 타겟을 좌우 (추론, 출처: 02_월간실적 1월·누적, 01_브랜드플랜).
- **예산 재배분이 멈춘 이유**: 본부장이 "가정의학 비중 근거가 약하다"며 보완 요구 → 전임자가 재회신하지 않음 (출처: 05_이메일, 04_업무메모).
- 캠페인 메시지 "심장을 지키는 첫 선택" / 시안 B안은 **확정 아님** ("거의 확정??", "안 정함") (출처: 01_브랜드플랜).
- 나머지 의사결정 배경 대부분은 자료에 명시되지 않음 → 전임자/상사 확인 필요 (`03_questions-to-ask.md`).
